# analise-estatistica-python

Projeto desenvolvido em Python para leitura e análise de números reais.
O programa recebe 15 números digitados pelo usuário e realiza cálculos
como soma, média, maior e menor valor, dobro, cubo e percentual de números ímpares.

## Funcionalidades
- Leitura de 15 números reais
- Cálculo da soma e média
- Identificação do maior e menor valor
- Cálculo do dobro e cubo dos números
- Cálculo do percentual de números ímpares
